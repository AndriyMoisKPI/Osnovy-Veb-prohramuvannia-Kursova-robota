Калькулятор теплоенергетика

Онлайн платформа для теплотехнічних розрахунків.

Призначення

Веб-додаток призначений для інженерів, студентів та фахівців галузі. Він дозволяє оперативно отримувати результати розрахунків стану газу, параметрів теплообмінників та гідравлічних втрат без використання складного ПЗ.

База даних

Додаток містить вбудований довідник фізичних властивостей матеріалів.
